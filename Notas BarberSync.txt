  Próximos pasos para tener el proyecto corriendo

  1. cp .env.example .env → configurar credenciales
  2. pip install -r requirements.txt
  3. Crear la base de datos PostgreSQL barbersync
  4. python manage.py migrate
  5. python manage.py createsuperuser
  6. Crear la primera Organization + Barbershop desde el admin
  7. python manage.py qcluster (para las notificaciones async)
